# Expense Tracker
## To run the code first download the zip file from repository and UnZip the folder in your prefered location

1. Open in your code editor like vs code
2. Then open up two terminals or split terminal
3. one for frontend and another for backend

# In terminal
## Frontend running
1. cd Frontend
2. npm install
3. npm start

## Backend running
1. cd backend
2. npm install
3. npm start

## frontend runs in url http://localhost:3000
## backend runs in url http://localhost:5000
